Welcome to GrailKit documentation!
==================================

This is documentation of GrailKit project - open source library of components used by Grail.
Source code available at http://bitbucket.org/grailapp/grail-kit

Features
--------

* Grail DNA I/O
* Bible formats parsing
* DMX protocol based on USB serial port
* OSC protocol implementation
* ArtNet protocol implementation
* MIDI protocol

Installation
------------

You can download source code from
http://bitbucket.org/grailapp/grail-kit

Install using pip

.. code-block:: bash

    pip install grailkit
